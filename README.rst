muckmuck
======

* track good and bad days

Installation
------------

* pull project

.. code-block:: bash

  git clone git@github.com:abondis/muckmuck.git
  cd PyPIM

* get virtualenv

.. code-block:: bash

  apt-get install python-virtualenv

* setup virtualenv

.. code-block:: bash

  virtualenv env
  . env/bin/activate

* install requirements

.. code-block:: bash

  pip install -r requirements.txt

* start the project

.. code-block:: bash

  python muckmuck.py
